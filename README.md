# 임서인의 자기소개

안녕하세요, 저는 조직의 맥락을 읽는 조율형 리더입니다. 

국어국문학을 전공하며 제가 진정으로 매료된 것은 문장 이면에 숨겨진 화자의 심리와 시대적 배경을 입체적으로 분석하는 과정이었습니다. 이러한 학문적 훈련은 저에게 사람의 마음을 깊이 있게 들여다보고, 복잡한 상황의 본질을 한발 앞서 꿰뚫어 보는 ‘현장 중심적 통찰력’이라는 귀한 선물을 주었습니다.

이러한 사고방식은 학생회 총괄로서 20여 명의 핵심 운영진을 통솔할 때 강력한 리더십의 원동력이 되었습니다. 저는 갈등의 표면적 현상에 매몰되기보다 팀원들의 실질적인 니즈와 숨은 의도를 파악하는 데 집중했습니다. 행사 기획 중 의견 충돌이 발생하면 각 팀원이 사용하는 언어 습관과 태도를 세밀히 관찰하여 그들이 진정으로 우려하는 지점을 찾아냈습니다. 덕분에 소모적인 감정싸움을 이성적이고 명확한 언어로 중재하며, 파편화된 의견을 하나의 목표로 단단히 응집시킬 수 있었습니다. 또한, 어떤 위기에서도 잃지 않는 밝고 긍정적인 에너지는 팀원들이 심리적 안정감을 느끼며 끝까지 완주하게 만드는 든든한 추진력이 되었습니다.

저는 상황의 맥락을 정확히 읽어내어 조직의 온도를 높이는 사람입니다. 업무 처리에만 그치지 않고 공동체에 필요한 언어적 배려가 무엇인지 고민하며, 세심한 관찰로 메우는 것이 저의 핵심 가치입니다.

앞으로 저는 구성원 모두가 최고의 역량을 발휘하도록 ‘심리적 안전망’과 ‘명확한 방향’을 동시에 제시하는 리더가 되겠습니다. 갈등의 맥락을 선제적으로 해소하고, 공동의 가치를 모두가 공감할 수 있는 언어로 공유하며 조직이 활기차게 성장하는 환경을 만드는 것이 저의 최종 목표입니다.
